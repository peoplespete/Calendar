Calendar
==================

[UNIX Cal](http://en.wikipedia.org/wiki/Cal_(Unix)) command line clone for years 1800-3000

Why?
-----------


Setup
------
Run from your command line:
```
git clone <CLONE_URL_FROM_GITHUB_PAGE>
cd Calendar
```

How To Use
-----------
Year View
```
ruby cal.rb YYYY
```
Month View
```
ruby cal.rb MM YYYY
```
